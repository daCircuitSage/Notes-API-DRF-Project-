#  Notes API - Django REST Framework

A simple and secure **Notes Management API** built with **Django REST Framework**.  
Each user can register, login, and manage their personal notes — Create, Read, Update, and Delete (CRUD) — using token-based authentication.

---

##  Features

 User Registration and Login (Token-based Authentication)  
 Create, Read, Update, and Delete Notes  
 User-specific data access (each user can only see their own notes)  
 Secure API with DRF Permissions  
 Clean, modular code structure — perfect for beginners  

---

## Tech Stack

- **Python 3**
- **Django 5**
- **Django REST Framework**
- **SQLite (default database)**
